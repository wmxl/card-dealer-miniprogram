# 小程序部署指南 - 私下使用版本

## 📋 部署流程概览

要让朋友私下使用小程序，有以下几种方式：

1. **开发版/体验版**（推荐）- 无需审核，快速分享给朋友
2. **正式版但不公开** - 需要审核，但不对外展示
3. **真机预览** - 临时测试使用

## 🚀 方式一：开发版/体验版（推荐，最简单）

### 优点
- ✅ 无需审核，立即可用
- ✅ 可以分享给最多50个体验者
- ✅ 适合小范围使用
- ✅ 可以随时更新

### 步骤

#### 1. 上传代码版本

1. 在微信开发者工具中，点击 **上传** 按钮
2. 填写版本号（如：`1.0.0`）
3. 填写项目备注（如：`桌游发牌助手 - 初始版本`）
4. 点击 **上传**

#### 2. 设置为体验版

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 登录你的小程序账号
3. 进入 **版本管理**
4. 找到刚才上传的开发版本
5. 点击 **选为体验版**
6. 体验版会自动生成二维码

#### 3. 添加体验者

1. 在 **版本管理** -> **成员管理** -> **体验成员**
2. 点击 **添加体验成员**
3. 输入朋友的微信号（需要朋友先关注你的小程序）
4. 或者直接分享体验版二维码给朋友
5. 朋友扫码后即可使用

#### 4. 分享给朋友

**方法1：分享二维码**
- 在版本管理页面，找到体验版二维码
- 保存二维码图片
- 分享到微信群或直接发给朋友

**方法2：分享链接**
- 体验版会自动生成链接
- 可以直接分享链接给朋友

### 注意事项

- 体验版最多支持50个体验者
- 体验者需要先关注你的小程序（通过二维码或链接）
- 每次更新代码后，需要重新上传版本并设置为体验版

## 🎯 方式二：正式版但不公开（适合长期使用）

### 优点
- ✅ 可以分享给更多人
- ✅ 更稳定，需要审核
- ✅ 可以选择不对外展示

### 步骤

#### 1. 完善小程序信息

1. 在微信公众平台 -> **设置** -> **基本设置**
2. 完善小程序信息：
   - 小程序名称
   - 小程序简介
   - 小程序头像
   - 服务类目（选择"工具"或"游戏"）

#### 2. 上传代码并提交审核

1. 在微信开发者工具中上传代码版本
2. 在版本管理中选择开发版本
3. 点击 **提交审核**
4. 填写审核信息：
   - 版本描述：`桌游发牌助手，用于朋友间玩桌游时随机分配字母`
   - 测试账号：可以填写你的微信号
   - 测试路径：`pages/index/index`
   - 测试说明：`这是一个工具类小程序，用于朋友间私下使用`

#### 3. 等待审核

- 审核时间通常1-7个工作日
- 审核通过后，会收到通知

#### 4. 发布但不公开

1. 审核通过后，在版本管理中点击 **发布**
2. 小程序会正式上线
3. **重要**：在 **设置** -> **基本设置** -> **小程序码及线下物料**
   - 可以选择 **不对外展示**（如果选项存在）
   - 或者不主动推广，只通过分享给朋友使用

#### 5. 分享给朋友

- 发布后，可以生成小程序码分享给朋友
- 朋友扫码后即可使用，无需添加为体验者

## 📱 方式三：真机预览（临时测试）

### 适用场景
- 临时测试
- 快速演示给朋友看

### 步骤

1. 在微信开发者工具中点击 **预览**
2. 使用微信扫码
3. 在手机上打开小程序
4. 可以分享给朋友，但每次预览链接有效期有限（通常24小时）

## 🔒 隐私和安全建议

### 1. 不公开小程序
- 不要在小程序商店公开
- 不要主动推广
- 只通过私聊或小群分享

### 2. 限制使用范围
- 使用体验版时，控制体验者数量
- 正式版可以通过分享控制使用范围

### 3. 数据安全
- 数据库权限设置为"仅创建者可读写，所有人可读"
- 定期清理无用数据
- 注意保护房间号，避免被他人加入

## 🎉 快速开始（推荐流程）

**最快的方式**：

1. **上传代码**
   - 微信开发者工具 -> 上传 -> 填写版本号

2. **设置为体验版**
   - 微信公众平台 -> 版本管理 -> 选为体验版

3. **分享二维码**
   - 保存体验版二维码
   - 分享到微信群或发给朋友

4. **完成！**
   - 朋友扫码即可使用

## 💡 常见问题

### Q: 体验版和正式版有什么区别？
A:
- 体验版：无需审核，最多50人，适合小范围使用
- 正式版：需要审核，无人数限制，更稳定

### Q: 朋友扫码后提示"需要添加为体验者"？
A: 在版本管理 -> 成员管理 -> 体验成员中添加朋友的微信号

### Q: 更新代码后朋友需要重新扫码吗？
A: 不需要，体验版会自动更新，朋友打开小程序即可看到新版本

### Q: 可以同时有多个版本吗？
A: 可以，但只能有一个体验版和一个正式版
